# Kaggle Learning Repository

Este repositorio reúne mi progreso y aprendizaje en **Kaggle**, incluyendo cursos, competiciones y certificaciones obtenidas a lo largo del camino en ciencia de datos y machine learning.

El objetivo principal es **documentar el proceso de aprendizaje**, servir como referencia personal y mostrar de forma ordenada las habilidades adquiridas.

---

## Estructura del repositorio

### `courses/`
En esta carpeta se encuentran los **cursos de Kaggle Learn** que he completado o estoy desarrollando.

Cada curso incluye:
- Cuadernos Jupyter (`.ipynb`)
- Explicaciones teóricas en Markdown
- Ejercicios prácticos
- Notas personales y ejemplos

Algunos temas tratados:
- Limpieza de datos (Data Cleaning)
- Pandas
- Machine Learning
- Visualización de datos
- SQL
- Feature Engineering

---

### `competitions/`
Aquí se documentan las **competiciones de Kaggle** en las que he participado.

Cada competición puede incluir:
- Análisis exploratorio de datos (EDA)
- Procesos de limpieza y transformación
- Modelos entrenados
- Resultados y conclusiones

Ejemplo:
- **Titanic: Machine Learning from Disaster**

---

### `badges-certificates/`
Esta carpeta contiene un **cuadernillo en formato Jupyter Notebook**, utilizado exclusivamente para **Markdown** (sin código), donde se recopilan:

- Badges obtenidos en Kaggle Learn
- Certificados de aptitud
- Títulos de los cursos completados
- Breve explicación de las habilidades adquiridas en cada uno

Este cuadernillo funciona como un **portafolio de certificaciones**, centralizando todos los logros en un solo lugar.

---

## Objetivo del repositorio

- Consolidar conocimientos en ciencia de datos
- Mantener un historial de aprendizaje estructurado
- Servir como portafolio técnico
- Facilitar la revisión y reutilización de notebooks y apuntes
